Te encontrarás con el código de las cosas que he trabajado si es que alguno lo encuentras interesante. Todo ha sido realizado por la página P5js.

Este te llevará al segundo trabajo a realizar en esta clase, puesto que la primera fue esto, crear la página web. Este segundo trabajo se trata de un autorretrato, el cual se hizo en base a figuras geométricas tales como fueron en este caso, rectángulos y círculos.

En este trabajo puedes crear un espiral, sin embargo, este se crea mediante los sliders que se ubican abajo del canvas. El espiral quedará distinto dependiendo de cuanta intensidad se le de al slider.

A este se le llama "Ciclo for", en este caso, si pasas el mouse por sobre el dibujo de los puntos, verás como la pantalla cambia de color, ¡cada uno más pastel que el anterior!

La "Recursión" es cuando un elemento se copia varias veces seguidas, en este caso, en mi trabajo, se repite muchísimas veces un rectángulo.

Aquí te encontrarás con unos "Átomos", los cuales van saltando de un lado para el otro de manera autónoma, por lo tanto, es algo digno de admirar.
